# 海報製作規則

這份規則用來檢查夏令營、育樂營、班會、課程活動等海報是否資訊完整。每張海報都要先確認「必要資訊」齊全，再處理美術風格與版面。

## 必要資訊

每張海報至少要包含以下內容：

1. 班會名稱或活動名稱
   - 例如：小天使育樂營、小小天使夏令營、明德新民法會。
   - 名稱要放在主標題位置，必須是第一眼能看到的文字。

2. 活動時間
   - 要寫清楚日期、星期、開始時間、結束時間。
   - 若有報到與賦歸時間，需分開標示。
   - 跨日活動要寫出每一天的日期，不只寫「兩天一夜」。

3. 活動地點
   - 要寫完整地點名稱。
   - 若有地址，也要放上完整地址。
   - 若地點不好找，可補充樓層、入口、集合點或導航關鍵字。

4. 課程內容
   - 至少列出 3 到 5 個主要課程或活動亮點。
   - 課程名稱要用家長和學員看得懂的說法，不只放內部代號。
   - 若有特別主題、分齡課程、闖關、手作、戶外活動，應優先呈現。

5. 參加對象
   - 要寫清楚適合年齡、年級或班別。
   - 若有分組，需標示各組對象。

6. 費用與公開聯絡資訊
   - 若文件有提供報名費、活動費、材料費、保證金或其他需繳費用，應標示清楚。
   - 個人姓名、個人手機、個人 Line ID 等個人聯絡資料不放入海報整理稿與海報成品。
   - 場地電話、官方窗口、單位公用電話或使用者明確要求公開的聯絡方式，才可作為聯絡資訊保留。
   - 「報名方式」一律不列為必要資訊；文件未提供時不可放「報名方式：待補」。
   - 「主辦/承辦」一律不列為必要資訊；文件未提供時不可放「主辦/承辦：待補」。

## 建議資訊

版面空間足夠時，可以加入以下資訊：

- 活動口號或主題句。
- 攜帶物品。
- 注意事項。
- 餐食、住宿、交通安排。
- QR Code。
- 家長說明會或行前通知時間。

## 版面優先順序

資訊擺放的優先順序如下：

1. 活動名稱
2. 活動日期與時間
3. 活動地點
4. 課程亮點
5. 參加對象、費用與公開聯絡資訊
6. 注意事項

如果版面太滿，先縮短說明文字，不可刪除活動名稱、時間、地點、課程內容、參加對象、費用與公開聯絡資訊。

## 文字規則

- 主標題要短、清楚、醒目。
- 日期時間要避免只寫簡稱，例如不要只寫「7/4-7/5」，應補上年份、星期與報到/結束時間。
- 地點不要只寫俗稱，需補上正式名稱或地址。
- 課程內容用短句呈現，每項最好不超過 12 個中文字。
- 不放個人聯絡資訊：個人姓名、個人手機、個人 Line ID 應移除；場地電話、官方窗口或單位公用電話可保留。
- 不使用「報名方式：待補」或「主辦/承辦：待補」這類佔位文字；若文件沒有資料就省略。

## Canva 可編輯檔規則

若要產生 Canva 設計檔，目標是提供可供人工微調的原生 Canva 版面，不只是輸出扁平圖片。

- 文字應使用 Canva 文字框，方便後續改字、調字距、換字型與移動位置。
- 色塊、資訊框、分隔線、按鈕式標籤等版面元素應使用 Canva 形狀或原生元素，方便移動、刪除與改色。
- 裝飾素材應盡量使用可獨立選取的元素，避免把標題、資訊與裝飾合併成單一圖片。
- 背景圖、AI 生成插圖或照片可以作為圖片素材，但不可承載唯一的重要文字資訊。
- 若使用 AI 生成圖片，應把它視為背景或主視覺素材；活動名稱、時間、地點、課程、參加對象、費用與公開聯絡資訊仍要另做成可編輯文字物件。
- 交付前需確認主要物件在 Canva 中可單獨選取、移動、刪除或編輯。

## Canva 權限與模板來源

本專案以使用者具備 Canva 團隊版、但不假設具備 Canva Enterprise 為前提。

- 可以使用使用者 Canva 帳號或團隊中已建立的設計、一般模板、元素、照片、插圖、文字框、形狀與協作資料夾。
- 不假設可使用 Enterprise 專屬的品牌模板 dataset、自動填入 autofill、企業級審核或進階權限控管。
- 若 Canva connector 的品牌模板搜尋或自動填入功能回覆需要 Enterprise，不可因此改成從空白 HTML 硬做正式候選稿；應改採「使用者在 Canva 選模板並建立成自己的設計，再由代理人編修」的流程。
- 使用者不需要把 project 設成公開共享。只要設計已在使用者 Canva 帳號或團隊權限內，代理人通常可透過 connector 讀取單一 design。若讀不到，再請使用者確認權限或重新貼設計連結。
- 不要要求使用者貼 `https://www.canva.com/projects` 專案列表頁。應請使用者貼單一設計連結，例如 `https://www.canva.com/design/...` 或 `https://canva.link/...`。
- 只要使用者已提供可讀取的 Canva design 或 canva.link，交付與 review 應以 Canva draft 預覽圖、draft 編輯連結、正式 Canva 設計連結為主，不再另外輸出 HTML 候選稿。
- Canva draft 階段必須直接在聊天中顯示預覽圖給使用者看；draft 編輯連結不可作為主要驗收依據，因為使用者端開啟時可能跳回一般編輯模式，看不到暫存修改。
- HTML 預覽只在沒有可用 Canva design、Canva connector 無法讀取、或使用者明確要求本機 HTML 預覽時才產生。

## 使用者已挑 Canva 模板時的優先流程

若使用者已在 Canva 挑好模板，並貼出單一 design 連結或 canva.link 短連結，優先採用本流程。此流程優先於「代理人自行搜尋 5 個模板」。

1. 讀取活動資訊與本規則。
   - 先整理活動名稱、日期時間、地點地址、課程亮點、參加對象、報名費用與公開聯絡方式。
   - 個人姓名、個人手機、個人 Line ID 不整理進海報稿；場地電話、官方窗口或單位公用電話可保留。
   - 若資料不足，必須標示待補，但不可自行編造日期、電話、地址、費用或課程名稱。
   - 報名方式與主辦/承辦一律不用整理、不用放入海報，也不可標示待補。

2. 解析使用者提供的 Canva 連結。
   - 若是 `canva.link` 短連結，先 resolve shortlink，取得目標 design URL 與 design ID。
   - 若是一般 Canva design 連結，擷取 design ID。
   - 若使用者貼的是 project 列表頁，請使用者改貼單一 design 連結。

3. 開啟 Canva editing transaction。
   - 讀取 richtexts 與 fills，確認哪些重要資訊是可編輯文字框。
   - 先回報目前讀到的設計名稱、design ID、頁數與可編輯文字概況。
   - 預覽圖若由工具回傳，必須在聊天中顯示給使用者。

4. 替換文字與微調排版。
   - 將模板文字替換成活動文件中的正確資訊。
   - 活動名稱、主題句、日期、報到/賦歸時間、地點、地址、對象、報名費用、公開聯絡資訊要優先放入。
   - 缺資料以「待補」標示，但報名方式與主辦/承辦除外；這兩項不用放，也不標待補。
   - 若版面空間不足，先壓縮說明文字，不刪必要資訊。
   - 盡量保持原模板美術風格，只調整文字層級、位置、寬度與字級，避免破壞主視覺。

5. 預覽與檢查。
   - 每次執行 Canva 文字替換、圖片替換、位置調整、字級調整或任何會影響版面的 editing operation 後，都必須主動取得並在聊天中顯示 draft 預覽圖，不可只提供 draft 編輯連結。
   - 修改後必須直接在聊天中顯示 draft 預覽圖；draft 編輯連結只能作為輔助資訊，不可要求使用者開 draft link 來驗收。
   - 回覆時必須明確說明「這是 draft，尚未 commit」或「這是正式已 commit 設計」，避免使用者誤以為原 Canva 設計已更新。
   - 明確列出已放入資訊與仍待補資訊。
   - 檢查是否有亂碼、假中文、錯字、文字框太窄、文字被素材遮住、重要資訊不存在於文字層等問題。
   - 自主檢驗不可只看工具回報 `success` 或 richtexts；必須以預覽圖確認畫面上看得到活動名稱、日期時間與地點。

6. 儲存或取消。
   - 未取得使用者明確回覆「儲存」、「存」、「commit」或同等意思前，不可 commit editing transaction。
   - 使用者要求先 commit 一版時，才可 commit；commit 後回覆正式 Canva 編輯連結與 design ID。
   - commit 後必須重新讀取正式 design 的內容或 metadata，確認更新已寫回正式設計；若正式 design 仍是原模板文字，必須立即告知使用者 commit 未達成預期，不可宣稱完成。
   - commit 後若工具可取得正式縮圖或預覽圖，應主動顯示給使用者作為正式版本示意。
   - 若使用者要求取消、重做或改用另一個模板，應 cancel 未儲存的 editing transaction，避免留下未處理 draft。

## Canva 海報工作流程

使用者提供活動資訊後，代理人需主動完成資訊整理、模板方向判斷、Canva 模板搜尋、5 個可編輯候選稿製作、文字檢查與預覽交付。使用者只負責最後 review，不要在一開始要求使用者先挑模板，除非使用者明確指定模板或素材。

1. 讀取活動資訊與本規則。
   - 先整理活動名稱、日期時間、地點地址、課程亮點、參加對象、報名費用與公開聯絡方式。
   - 個人姓名、個人手機、個人 Line ID 不整理進海報稿；場地電話、官方窗口或單位公用電話可保留。
   - 若資料不足，優先使用活動文件已有資訊，不可自行編造日期、電話、地址、費用或課程名稱。
   - 報名方式與主辦/承辦一律不用放入海報，也不用列為待補資訊。

2. 自行判斷模板方向。
   - 依活動類型、目標讀者、資訊密度與氣質挑選合適風格。
   - 兒童育樂營、夏令營優先採用明亮、童趣、親子友善、資訊清楚的版面。
   - 法會、班會、課程或正式活動應依主題改用較沉穩或莊重的版面。
   - 資訊多的活動應選擇有清楚資訊區、底部資訊帶或多欄資訊卡的模板，不選只適合少量文字的視覺型模板。

3. 搜尋 Canva 模板並挑選 5 個方向。
   - 做海報一定要先從 Canva 搜尋合適模板，不可跳過模板搜尋直接從空白畫布開始。
   - 搜尋關鍵字需依活動類型與受眾調整，例如夏令營、兒童活動、親子活動、課程、法會、營隊、活動海報。
   - 至少挑出 5 個適合的可用模板，優先選擇文字區清楚、資訊容量足夠、可編輯性高的模板。
   - 若 Canva 搜尋結果都不適合，才可用最接近的模板改版；仍需保留「從 Canva 模板出發」的流程。
4. 產生 5 個 Canva 可編輯候選稿。
   - 必須用前一步挑出的 5 個模板分別做出 5 個 Canva 可編輯海報候選稿。
   - 不可只比較 5 個模板後只製作 1 個海報，也不可用 1 個模板改出 5 個表面不同但結構相同的候選稿。
   - 若使用者另行指定候選稿數量，依使用者指定數量執行；未指定時一律產生 5 個。
   - 生成 prompt 必須明確要求繁體中文、可編輯文字框、完整必要資訊、避免亂碼與假中文。
   - 回覆候選稿時應提供可分辨的縮圖或預覽，不只提供容易混淆的候選連結。

5. 檢查 5 個候選稿的文字層。
   - 每個候選稿建立為 Canva design 後，都必須開啟 Canva editing transaction，讀取 richtexts，逐項確認文字內容。
   - 若有亂碼、假中文、錯字、缺漏資訊、文字框太窄或字太小，必須先修正。
   - 主要活動資訊必須是可編輯文字，不可只存在於圖片裡。
   - 若候選稿主要資訊無法轉成可編輯文字，該稿不可交付為合格候選稿，需換模板或重做。

6. 預覽與等待 review。
   - 5 個候選稿修改完成後，必須輸出可分辨的 Canva 預覽圖或可檢視成果，並提供各自的設計連結。
   - 明確告知每個候選稿目前是否仍為 draft。
   - 未取得使用者明確同意前，不可 commit 任何 editing transaction。

7. 儲存到 Canva。
   - 使用者明確回覆「儲存」、「存」、「commit」或同等意思，並指出要儲存哪一個候選稿後，才可 commit 對應的 editing transaction。
   - commit 後回覆正式 Canva 編輯連結與 design ID。
   - 未選中的候選稿若仍有開啟的 editing transaction，應 cancel editing transaction，不要留下未處理的 draft。
   - 若使用者要求取消或重做，應 cancel editing transaction 或另開新設計，不要留下未處理的 draft。

- [ ] 有報到時間或集合時間。
- [ ] 有活動地點。

- [ ] 有 3 到 5 個課程內容或活動亮點。
- [ ] 有參加對象。
- [ ] 若文件有提供報名費、活動費、材料費、保證金或其他需繳費用，已放入費用資訊。

- [ ] 重要文字沒有被圖案遮住。
- [ ] 手機上縮小觀看仍能讀到活動名稱、時間、地點。

## 教科書範例

參考範例：`example/115年學界明德新民法會_海報v2.png`

這張海報可作為本專案的標準範本，原因如下：

- 主標題非常明確：活動名稱「學界明德新民法會」放在最醒目的位置。
- 層級清楚：上方是年度、區域與活動名稱，中間是主視覺與主題句，下方是課程亮點，最底部是時間與地點。
- 必要資訊完整：活動名稱、日期、星期、開始時間、結束時間、地點名稱與地址都有出現。
- 課程亮點好讀：用 4 個短句呈現重點，例如共生共好、純陽壇巡禮、百善孝為先、信愿行。
- 視覺服務資訊：飛機、道院、人物、星星都有主題感，但不會蓋住重要文字。
- 底部資訊帶很實用：時間與地點集中放在同一區，縮小看仍能辨識。

後續夏令營或育樂營海報應盡量採用同樣的資訊邏輯：

1. 最上方放活動名稱。
2. 中間放主視覺與一句主題句。
3. 中下方放 3 到 5 個課程亮點。
4. 最下方固定放日期、時間、地點與地址。
